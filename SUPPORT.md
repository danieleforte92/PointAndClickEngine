# Support

For reproducible defects, use the bug-report template. For feature ideas, use
the feature-request template and describe the authoring or runtime problem
before prescribing a solution. Community support is best effort; Creator Alpha
does not offer guaranteed response times or private production support.

For security-sensitive problems, follow [SECURITY.md](SECURITY.md) instead of
posting credentials, private projects, or exploit details in a public issue.

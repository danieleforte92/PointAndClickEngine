# Code of Conduct

We want a welcoming, constructive project for creators and contributors.
Participate with respect, assume good faith, and focus feedback on the work.

Harassment, discrimination, threats, deliberate intimidation, doxxing, and
sharing another person's private information are not acceptable. Maintainers may
remove content, limit participation, or take other proportionate action to keep
the project safe.

Report concerns through the private security route in [SECURITY.md](SECURITY.md)
when available, or through a minimal, factual issue if no private route exists.
Do not include secrets or sensitive personal information in public reports.

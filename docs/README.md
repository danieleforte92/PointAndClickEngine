# Documentation

Point & Click Engine documentation is organized by the reader's goal. Start
with the tutorial if you are learning the engine, use a how-to for a specific
authoring task, consult reference pages for exact formats and commands, and
read the explanations for architectural context.

## Current status

Creator Alpha `0.4.0-alpha.3` is a Windows-first, local-first release for
layered 2D adventures. The sample and starter projects validate cleanly, static
web export is available, and the editor is undergoing a modularization and
visual convergence initiative before the beta milestone.
